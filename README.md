# 🐘 Lista de Exercícios PHP

Projeto desenvolvido para a disciplina de **Programação Web**, utilizando **PHP**, **HTML** e **CSS**.

O objetivo do projeto é praticar os principais conceitos básicos da programação em PHP, incluindo:

- Algoritmos sequenciais
- Estruturas de decisão (`if` e `switch`)
- Laços de repetição (`for`, `foreach`)

---

# 📚 Estrutura dos Exercícios

Os exercícios estão organizados em três categorias:

## 1 Algoritmos Sequenciais

1. Conversor de moedas  
2. Cálculo de média  
3. Consumo de combustível  

---

## 2 Estruturas de Decisão (IF / SWITCH)

4. Maior número  
5. Verificação de idade  
6. Classificação de nota  
7. Dia da semana  
8. Verificação de número par ou ímpar  
9. Mês por extenso  

---

## 3 Laços de Repetição

10. Fatorial  
11. Somatório  
12. Números pares em um intervalo

## 4 Arrays
13. Média de 5 notas  
14. Média de 5 notas  
15. Lista de compras  

---


# 🖥️ Interface

O projeto possui uma interface simples e moderna com:

- Layout em **cards**
- **Glassmorphism**
- **Gradiente animado**
- Design **responsivo**

Cada exercício possui sua própria página com formulário para entrada de dados e exibição do resultado.
